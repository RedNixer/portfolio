# Design System - Portfolio Professionale

## Filosofia del Design
- **Minimalismo Elegante**: Pulizia visiva con focus su contenuti
- **Modernità**: Design contemporaneo con linee pulite
- **Professionalità**: Aspetto affidabile e competente
- **Accessibilità**: Facile navigazione e lettura

## Palette Colori

### Colori Principali
- **Background**: #FAFAFA (Bianco sporco)
- **Primary**: #1A1A1A (Nero carbone)
- **Secondary**: #404040 (Grigio scuro)
- **Accent**: #E67E22 (Arancione vibrante)

### Colori Supporto
- **Light Gray**: #F5F5F5
- **Medium Gray**: #CCCCCC
- **Success**: #27AE60
- **Warning**: #F39C12

## Tipografia

### Font Principali
- **Display**: 'Montserrat', sans-serif (Titoli)
- **Body**: 'Inter', sans-serif (Testo normale)
- **Accent**: 'Playfair Display', serif (Citazioni/testi speciali)

### Gerarchia Tipografica
- **H1**: 3.5rem (56px) - Montserrat Bold
- **H2**: 2.5rem (40px) - Montserrat SemiBold
- **H3**: 2rem (32px) - Montserrat Medium
- **H4**: 1.5rem (24px) - Montserrat Medium
- **Body**: 1rem (16px) - Inter Regular
- **Small**: 0.875rem (14px) - Inter Regular

## Componenti UI

### Navbar
- **Altezza**: 80px
- **Background**: rgba(255, 255, 255, 0.95) con blur
- **Shadow**: Subtle drop shadow
- **Logo**: 40px height

### Buttons
- **Primary**: Background #E67E22, text white
- **Secondary**: Border #E67E22, text #E67E22
- **Hover**: Transform scale(1.05) + shadow

### Cards
- **Border Radius**: 12px
- **Shadow**: 0 4px 20px rgba(0,0,0,0.1)
- **Hover**: Elevated shadow + translateY(-4px)

### Form Elements
- **Input Height**: 48px
- **Border**: 1px solid #CCCCCC
- **Focus**: Border #E67E22, shadow accent
- **Radius**: 8px

## Effetti Visivi

### Animazioni
- **Transition**: 0.3s cubic-bezier(0.4, 0, 0.2, 1)
- **Hover**: Scale, translate, shadow changes
- **Scroll**: Parallax subtle (max 20px)

### Background
- **Pattern**: Subtle dots grid (opacity 0.03)
- **Gradient**: Linear gradient su hero section
- **Shapes**: Geometric accent elements

## Layout

### Container
- **Max Width**: 1200px
- **Padding**: 20px mobile, 40px desktop
- **Grid**: 12-column system

### Spacing
- **Section**: 80px vertical spacing
- **Elements**: 24px gap standard
- **Components**: 16px internal padding

## Responsive Breakpoints
- **Mobile**: 320px - 768px
- **Tablet**: 768px - 1024px
- **Desktop**: 1024px+
